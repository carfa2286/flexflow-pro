# Instalar FlexFlow Pro en iPad (PWA)

## Requisitos
- iPad con Safari
- Misma red WiFi que tu Mac
- FlexFlow Pro corriendo en Mac

## Instalación Rápida (3 pasos)

### Paso 1: Obtén IP de tu Mac
Ejecuta en terminal de Mac:
```
ifconfig | grep inet
```

### Paso 2: Abre en iPad Safari
- Dirección: http://[tu-ip-local]:3000
- Ejemplo: copiar la IP que obtuviste

### Paso 3: Agregar a Pantalla de Inicio
1. Toca Compartir (abajo derecha)
2. Busca "Agregar a pantalla de inicio"
3. Dale nombre y confirma

✅ Listo! Funciona como app nativa + offline

## Beneficios PWA
- Sin necesidad de App Store
- Funciona sin internet
- Sincroniza automáticamente
- Acceso instantáneo
