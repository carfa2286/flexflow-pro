## 🚀 Getting Started

### Requirements
- Node.js 16+
- npm or yarn
- Modern web browser

### Installation

1. **Clone & Install**
```bash
cd loan-app
npm install
cd client
npm install
```

2. **Start Backend**
```bash
npm start  # from loan-app directory
# Runs on http://localhost:3001
```

3. **Start Frontend**
```bash
cd client
npm start  # from loan-app/client
# Opens http://localhost:3000
```

### 📱 On iPad/Tablet
See `INSTALL_IPAD.md` for PWA installation on iOS devices.
